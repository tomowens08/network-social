Paramétrage et personnalisation

phpFreeChat a été développé avec un objectif de simplicité et de flexibilité.

La mise en oeuvre est relativement simple car tous les paramètres sont centralisés dans un tableau et passé en paramètre au moment de l'initialisation de l'instance de l'objet phpFreeChat.

Paramétrer l'apparence (couleurs, tailles ...)

Pour cela, vous devez créer un nouveau thème que vous allez personnaliser (par exemple appelez le montheme) :

  1. Créez un répertoire : phpfreechat/themes/montheme/
  2. Créez une nouvelle feuille de style CSS : phpfreechat/themes/montheme/templates/style.css.tpl.php
  3. Insérez les règles CSS que vous voulez dans ce fichier. Par exemple pour changer la couleur de la date et de l'heureen rouge red : 
    span.<?php echo $prefix; ?>heure, span.<?php echo $prefix; ?>date {
  color: red;
}

    Remarque : utilisez <?php echo $prefix; ?> pour préfixer toutes les classes et ids car le prefix peut être amené à changer dans le future.

  4. Réglez le paramètre theme dans votre chat : 
    $params["theme"] = "montheme";
